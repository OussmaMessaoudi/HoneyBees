# HoneyBees

HoneyBees is a cross-platform mobile app built with Flutter.  
It allows users to share their bandwidth in exchange for money.  

<img width="1100" height="619" alt="honeybees" src="https://github.com/user-attachments/assets/c8d34790-ab3f-4e57-bc83-6ffa4cd8fec5" />
## Features
- Bandwidth sharing via integrated SDK
- User authentication with Firebase Auth
- Cloud storage with Firebase Storage
- Local storage using SharedPreferences
- Clean and simple user interface
- Android-first focus

## Technology Stack
- Flutter
- Firebase Auth
- Firebase Storage
- SharedPreferences
- Bandwidth Sharing SDK integration
